# Rotate Matrix

Given a 2D square matrix, rotate it by 90 degrees clockwise. 

**Note:** Solution has to have constant space complexity $$O(1)$$

## Example

```
Input:
[
    [1, 2],
    [3, 4]
]
Ouput:
[
    [3, 1],
    [4, 2]
]
```
