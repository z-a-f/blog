# Spiral (Example)

Given a matrix of $$m\times n$$ elements, where $$m$$ is the number of rows and $$n$$ is the number of columns, return all elements in a **spiral** order.

<!--sec data-title="Example" data-id="example" data-show=true ces-->
##### Input:
```
[
    [ 1, 2, 3 ],
    [ 4, 5, 6 ],
    [ 7, 8, 9 ]
]
```
##### Output:
```
[1, 2, 3, 6, 9, 8, 7, 4, 5]
```
