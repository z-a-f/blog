# Find Tripple Repeating

Given a *read-only* array of $$n$$ integers, find out if any element occurs more than $$n/3$$ times.

If there is no solution return `-1`.

## Requirements
$$T = O(n) \\
S = O(1)$$

## Example

```
Input: [1, 2, 3, 1, 1]
Output: 1
```

### Explanation

1 repeats three times, which is more than 5/3.
